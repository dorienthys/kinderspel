Opdrachten

Hoe start ik een nieuw verhaal?▼

Bij 'mijn verhalen' kan je kijken welk verhaal je nu bent gestart. Als je een nieuw verhaal starten wil klik je op één van de andere verhalen op deze pagina.

Hoe koop ik een nieuwe opdracht?▼

Sommige opdrachten zijn gratis, voor anderen moet je aan je ouders vragen deze te like'n op Facebook of deze via de App store aan te schaffen. Vraag dus je mama of papa om hulp.

Waarom staat er een Facebook knop?▼

Jouw mama of papa kan zich via Facebook aanmelden, nadien kunnen ze dan jouw gemaakte opdrachten op facebook posten of dit spel like'n op Facebook. Nadien kan jij dan nog meer opddrachten en spelletjes spelen.

Hoe stijg ik een level?▼

Als je opdrachten vervolledigd en deze laat goedkeuren door je ouders krijg je XP punten. Hoe meer opdrachten je vervolledigd, hoe sneller je punten krijgt en dus stijgt naar een hogere level.

Voor de ouders

Hoe verloopt de aankoop van een opdracht?▼

Sommige opdrachten zijn gratis, voor anderen hoef je enkel onze facebook-pagina te like'n of iemand te introduceren voor deze game. De aankoop van nieuwe opdrachten verloopt via de App Store of Play Store. Het is dus een veilige manier van aankoop en verloopt net zoals de aanschap van een nieuwe applicatie voor je smartphone of tablet.

Wat gebeurt er met mijn Facebook account?▼

Da's KinderSpel zal nooit zelf persoonlijke informatie verspreiden of posten op Facebook. Je kan via de app gemaakte opdrachten posten op Facebook

Hoe voeg ik een kind-account toe?▼

Ga via de navigatie (linksboven) naar je ouderaccount, van daaruit kan je je kindaccounts aanpassen of er extra toevoegen.
